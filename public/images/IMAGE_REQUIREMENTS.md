# Image Requirements

This directory should contain the following images for the portfolio website:

## Required Images

### Case Study Images
- `eurovision-arena.jpg` - Eurovision 2023 at M&S Bank Arena
- `creamfields-festival.jpg` - Creamfields Festival scene
- `rebellion-festival.jpg` - Rebellion Festival venue/image
- `ai-platform-mockup.png` - AI Event Management Platform mockup/screenshot

### Page Hero Images
- `portfolio-hero.jpg` - Hero image for Portfolio page (event montage)
- `headshot.jpg` - Professional headshot for About page

### Gallery Images (Optional)
- `gallery-1.jpg` - Major Event Operations
- `gallery-2.jpg` - Team Leadership
- `gallery-3.jpg` - Control Room Operations

## Organization Logos (Optional)

Create a `logos/` subdirectory with:
- `liverpool-fc.png` - Liverpool FC logo
- `g4s.png` - G4S logo
- `bbc.png` - BBC logo
- `ha-protection.png` - H&A Protection logo
- `compact-security.png` - Compact Security logo

## Image Specifications

- **Format**: JPG for photos, PNG for logos/screenshots
- **Recommended Size**: 
  - Hero images: 1920x1080px (16:9 aspect ratio)
  - Case study images: 1200x800px
  - Gallery images: 800x600px
  - Headshot: 600x600px (square)
  - Logos: 200x100px (maintain aspect ratio)
- **Optimization**: Compress images before uploading to reduce load times
- **Alternative**: Use WebP format for better compression

## Notes

- All images will be displayed with object-fit: cover for consistent sizing
- Missing images will show gray placeholders but won't break the site
- Ensure all images are properly licensed for use
